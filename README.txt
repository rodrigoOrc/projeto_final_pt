Para executar o projeto, é necessário a instalação das seguintes dependências:

# 1. Baixar o Vite:
(O Vite facilita a implementação de API no projeto em que ele for inserido)
	npm create vite@latest 

# 2. Após escrever o nome do projeto, selecione "React" e Javascript, e siga as instruções do Vite.

# 3. Instalar as dependências necessárias utilizadas dentro do projeto:
	npm install react-icons react router-dom
